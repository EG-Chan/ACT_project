# PredictUp
한가람 교육센터 최종프로젝트

1. 프로젝트 배경

틱톡에서 인기를 끌고 있는 노래들이 가끔씩 빌보드 차트에 진입하고 있습니다. 그러나 빌보드 100위에 들어가는 곡들보다 훌륭한 곡들이 차트에 진입하지 못하는 현상을 관찰하였습니다. 이러한 차트 진입 여부의 차이점을 밝히기 위해 노래 분석을 수행하고자 합니다.
